词库素材文件夹
================

把视频相关文本放进此文件夹，程序启动或「刷新词库」时会自动合并进词库。

支持格式：
  - .srt / .vtt / .ass  字幕文件（推荐：从视频导出字幕后放入）
  - .txt                纯文本，每行一个词或一句话
  - .json               格式：{"words":[{"word":"词","meaning":"释义","hint":"来源"}]}

从视频提取字幕（需本机安装 ffmpeg）：
  ffmpeg -i 你的视频.mp4 -map 0:s:0 字幕.srt

进阶：运行同目录上一级的 extract_vocab.py 可批量扫描并生成 vocab.json。
