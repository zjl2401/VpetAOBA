# 桌面桌宠 v1.2 功能规格

本文档汇总已实现需求，供后续维护对照。

## 菜单结构

| 模块 | 子项 |
|------|------|
| **模式** | 跟随、自由走动、睡眠、游戏、音乐 |
| **面板** | 打开面板、莱姆/邀请/宠物（预留）、暴露 |
| **互动** | 动作 ▶、表情 ▶ |
| **动作** | 打招呼、下蹲、打电话、吃东西、睡眠、工作、侧踢、×生活、是/否、判断 |
| **表情** | 开心、生气、疑问、伤心、有主意、脸红、wink、点赞 |
| **系统** | 大小设置 ▶、AI 对话、日程提醒、音乐设置、召唤小人、退出 |

- Esc：退出任意状态回到自由走动（菜单中不显示此项）
- **模式切换优先于**睡眠 30s、工作、互动等，切换时打断当前行为

## 对话框与音效

- 所有对话框文字：**逐字打字动画** + `type_cache.wav` 打字音
- 无对话动作：wink、脸红、有主意、下蹲
- ×生活文案：`我只是像素哦，更多精彩内容请在正版游戏《戏剧性谋杀》中解锁`（末尾仅一个「哦」）

## 表情与特效

| 动作 | 特效 | 对话 |
|------|------|------|
| 开心 | 跳跃 + **像素小花**（身后） | 无 |
| 生气 | walkback 121212 序列期间**持续向上平移** + 像素怒 mark | 有 |
| 伤心 | 头上**像素下雨**动画 | 有 |
| 有主意 | 左上角**灯泡点亮**动画 → eat2 | 无 |
| 侧踢 | 粒子特效，**持续 3s** 后结束 | 有 |
| 音乐模式 | 脚下**像素声波**，随播放进度律动 | 有 |
| 其他互动 | 无背景粒子（除上述） | 按配置 |

## 判断 / 暴露

- **判断**：stand + 对话框「你所问问题的答案是」5s → 随机 yes/no 揭晓
- **暴露**：无黑屏；小人旁圆环（1/6 蓝、其余白），指针旋转，**Enter** 判定；**成功/失败用打字对话框**显示

## 拖拽与 move

- 拖拽 move23 动画结束后松手：小人**整体下移**模拟落地（可配置位移）
- 游戏模式：**鼠标移动**控制小人位置（非拖拽）

## 游戏模式

- 时长 **30 秒**
- 鼠标移动小人接掉落食物
- 接住的食物写入 **背包**（`food_inventory.json`，多实例共享累积）
- HUD 显示倒计时、接住/错过、库存

## 背包 / 食物

- 面板展示：像素图标、数量、**体力/心情恢复值**
- 食物种类见 `FOODS` 常量（面包、苹果、蛋糕等）
- 吃东西：选食物 → **拖拽给小人**喂食；恢复对应体力/心情

## 跟随模式

- 跟随鼠标走动
- 短时间内频繁变向（默认 2.5s 内 8 次）：stand **3s** + 打字框「我晕了……」→ 恢复跟随

## 睡眠

- 互动睡眠：深度睡眠 **30s** 后自动醒来
- 模式切换/Esc 可打断

## 工作模式

- 搬运箱子任务；拖拽小人后松手**继续工作**（不卡死）

## 音乐模式

- 仅自由模式可用
- 循环 BGM + music 精灵 + 像素声波（尽量与 `pygame.mixer.music.get_pos()` 同步）

## 面板定位

- 展开时若超出屏幕，向**不靠屏幕外**的方向展开（右 → 左 → 下）

## 持久化

- `food_inventory.json`：食物库存（跨进程共享）
- `schedules.json`、`ai_config.json`、`music_config.json`
