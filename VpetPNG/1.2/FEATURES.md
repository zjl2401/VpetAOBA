## 已实现功能

### 基础与菜单
- 四大模块：模式 / 面板 / 互动 / 系统（右键打开）
- 左键拖拽移动、透明抠图精灵、三档桌宠大小（持久化）
- 字体大小多档（持久化）
- **操作说明**（系统菜单 / F1）：专题索引 + 首次进玩法提示一次
- **一次性提示**：游戏、伴侣、音乐、莱姆邀请等
- **难度** 低/中/高：影响体力/心情、采集、暴露、莱姆对战
- **重置**：恢复初始设置（编号保留）
- **关于** / **问题反馈**（GitHub / 小红书 / bilibili）

### 互动
- 动作：打招呼、下蹲、电话、喂食、睡眠、运送、侧踢、是/否、判断、召唤金目等
- 表情：开心/生气/疑问/伤心/有主意/脸红/wink/点赞等（含背景特效）
- 连点台词、动作结束自动清理特效
- 视觉规格细节以 `REQUIREMENTS.md` / `IMMUTABLE_FILES.md` 为准

### 面板
- 体力 / 心情 / 食物背包
- **智能伴侣（莲）**：跟随；工作模式可导航
- **莱姆**：练习对战（本地）；邀请对战（占位）
- **暴露 QTE**（Enter 判定）

### 系统
- **我的**：编号、日记、日程、回忆（画廊/留声）、成就等
- **设置**：大小、字体、声音、显示层级；桌宠编号只读展示（自动分配）
- AI 对话、天气预报、重置、关于、反馈、退出

### 语音与音乐
- 场景 / 随机语音（`voice_system`）；音乐模式下不播语音
- BGM 漫步 + 音游谱面缓存
- 声音设置可开关与资源检查

### 编号
- `PET_ID_FEATURE=True`：启动后按设备指纹**自动分配**桌宠编号；同机找回；玩家不可手填
- 拷贝存档：沿用原编号（shared）
- 服务端：`cloud/pet_id_server.py`、Cloudflare Worker（`policy: auto_assign`）
- 无云端时本机分配临时号

### 快捷键
| 按键 | 功能 |
|------|------|
| Esc | 退出当前玩法 / 子窗口 |
| F1 | 操作说明 |
| Ctrl+Shift+H / E / T / J / N | 打招呼 / 喂食 / 电话 / 下蹲 / 睡眠 |
| Ctrl+Shift+A / V / Q | AI 对话 / 主菜单 / 强制退出 |
| Enter | 暴露判定 |
| D F J K | 音游按键 |

## 必做维护（仓库卫生）

1. **只保留最新正式包**：`release/Vpet_build_*` 打包后自动只留最近若干份；勿提交到 Git（已在 `.gitignore`）。
2. **勿把 `_old` 媒体编进包**：正式资源用 `bundled/Vpetmusic`、`bundled/Vpetvoice`；备份目录勿再拷进 release。
3. **功能清单以本文为准**：大功能落地后更新本表；动画细节不改 `REQUIREMENTS.md` 既有条目。
4. **发布前自测**：自由/漫步/跟随/睡眠/工作/音乐；采集/打字/背单词/音游/RPG；面板伴侣与暴露；编号激活（有服务时）。
