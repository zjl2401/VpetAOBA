# Vpet 1.2 功能清单与实现状态

> 开发者：翛然而往

## 已实现功能

### 基础与菜单
- 四大模块：模式 / 面板 / 互动 / 系统（右键打开）
- 左键拖拽移动、透明抠图精灵、三档桌宠大小（持久化）
- 字体大小四档（小/中/大/特大，持久化）
- **操作说明**（系统菜单 / F1），首次启动自动弹出一次
- **一次性提示**：游戏模式、智能伴侣、音乐模式、莱姆邀请等（只显示一次）
- **难度** 低/中/高（系统菜单）：影响体力/心情下降、接食物、暴露 QTE、莱姆对战
- **重置** 恢复初始设置（编号保留）
- **关于** 苍叶百科风文案 + 《戏剧性谋杀》正版获取指引

### 模式
| 模式 | 说明 |
|------|------|
| 跟随 | 跟鼠标，同向绕圈 2 圈晕眩 |
| 自由 | 随机动作/表情/词库对话 |
| 漫步 | 仅 stand/walk |
| 睡眠 | 深度睡眠与唤醒 |
| 游戏 ▶ | 子菜单：**采集**（原游戏）、打字、背单词、刷新词库 |
| 音乐 | BGM + music 精灵 + 底部律动 |

### 小游戏（模式 → 游戏 ▶）
| 游戏 | 说明 |
|------|------|
| **采集** | 30 秒接食物（难度影响速度/判定），记录写入 `pet_profile.json` |
| **打字** | 30s 倒计时，按得分 C~S 评级；中/英/日词库；虚拟键盘 + 下一键闪光 |
| **背单词** | 英/日可选；**自由模式每日最多 3 次** |

### 互动
- 动作：打招呼、下蹲、电话、喂食、睡眠、工作、侧踢、是/否、判断、**召唤金目**
- 表情：开心/生气/疑问/伤心(分阶段+雨，无云)/有主意/脸红/wink/点赞
- **点赞** 背景像素发光；**wink** 爱心声波像素
- **连点对话**：各状态/动作连点触发额外台词
- 动作结束自动清理背景特效（雨/灯泡/点赞/wink 等）
- 睡觉互动 30s、move 落地、生气 walkback 等细节

### 面板
- 体力/心情/食物背包
- **智能伴侣（金目）**：尺寸 100px；侧向跟随；游戏模式快速跟紧；**工作模式在桌宠前方导航**（「我来导航/我来规划路线」）；音乐律动
- **莱姆 ▶** 练习对战 / 邀请对战（占位）
- **暴露 QTE**（Enter 判定，难度影响扇区/指针速度）

### 系统
- **我的 ▶**：桌宠编号、日记、日程提醒
- **设置 ▶**：大小、字体、声音
- AI 对话、重置、关于、退出

### 莱姆
- **练习对战**：本地回合制（攻击/防御/必杀），难度影响伤害
- **邀请对战**：入口已留，显示编号与说明（见下方「未实现」）

### AI
- 千问 DashScope 默认配置 + 多轮记忆
- 无 Key 时本地 fallback

### 词库
- 内置词库 + `vocab_sources/` 字幕/文本
- `extract_vocab.py` 批量生成

### 编号与记录
- `PET_ID_FEATURE=True` 时生成 `pet_profile.json` 桌宠编号
- 同一编号下 **采集 / 打字 / 背单词** 等记录统一存入 `records`（各保留最近 120 条）
- 改为 `False` 打包无编号版

## 暂未实现 / 需额外工程

| 功能 | 原因 | 建议 |
|------|------|------|
| **莱姆真·联机对战** | 需 WebSocket/UDP 服务器、房间匹配、状态同步、 NAT 穿透 | 自建 relay 或用 Steam/第三方大厅；客户端只留了编号展示 |
| **跨电脑邀请** | 同上，无后端 | 可先局域网原型 + 固定 IP 中继 |
| **视频自动语音识别抽词** | 需 Whisper 等依赖与算力 | 已支持手动导出字幕后放入 `vocab_sources/` |
| **莱姆角色独立精灵** | 无 rhyme 素材 | 当前对战为 UI + 数值，可后续加立绘 |
| **游戏记录 UI 查看** | 仅后台存储 | 可在「我的」中加历史记录页 |
| **云端存档/编号登记** | 无服务器 | 编号与记录仅本地 `pet_profile.json` |
