# Silent Oath — 骑士救公主

用现有像素素材做的小型 2D RPG：操控骑士穿越地图救出公主，支持随机地图与 DIY 编辑。

## 快速开始

```bash
# 1. 抠图 + 统一尺寸（输出到 assets/）
python process_assets.py

# 2. 启动游戏
python game.py
```

依赖：`pygame`、`Pillow`（已装可忽略）

```bash
pip install pygame Pillow
```

## 新特性

- **大地图 + 镜头跟随**：关卡远大于屏幕，视角跟着骑士走
- **地面 / 地下**：踩楼梯切换楼层；地下以 **brick 砖地** 为背景、**rock 岩石** 为隔断，且略暗
- **4 关冒险**：前 3 关找地下金色关卡门进入下一关；最终关在地下救出公主
- **DIY**：支持 Tab 切层、画楼梯（自动双层同步）、方向键/滚轮平移大地图

## 操作

| 场景 | 按键 |
|------|------|
| 菜单 | ↑↓ / WS 选择，Enter 确认 |
| 冒险 | WASD 移动，踩楼梯切换地面/地下，Esc 回菜单，R 重开本关 |
| 过关/通关 | Enter 继续 |
| DIY | 左键画、右键擦；Tab 切层；方向键/滚轮平移；`1`–`0`/`-`/`=` 笔刷；`S` 起点；`G` 终点；`Ctrl+S` 保存；`P` 试玩 |

## 文件

- `process_assets.py` — 去掉外圈白/黑/绿背景，角色 48×64，地砖/道具 48×48
- `game.py` — 游戏主程序
- 根目录 `*.png/*.jpg` — 原始素材
- `assets/` — 处理后的素材
- `maps/` — DIY 地图 JSON（首次保存后出现）
